识别扇叶，给出扇叶的特征五点二维坐标（图像坐标系下）

识别策略

给出了要极大的扇叶，我们就找出在这片扇叶角度周围的所有T和moons
然后先匹配两个moons，再匹配一个T一个moon。这里优先找出最外侧的moon，然后再找出次外侧的moon，然后再找出T，最后再找出最内侧的moon。

返回点的顺序

以最外侧开始随机给定0-1，然后内测同样顺序给定2-3.如下图

更新状态机状态，使用之前掩膜出来的流水灯

函数过程

首先对rect进行自定义排序，找到距离每片扇叶角度最近的那个，然后判断是否在范围内，如果在范围内，就更新状态，如果不在范围内，就判断是否需要重置，如果需要重置，就重置，否则就更新状态。

如何判断是否需要重置

  • 如果 fan_num == 1 或 fan.start_angle==-1(刚启动)：
    • 因为此时只有一片扇叶，可能是刚启动，也可能是刚重置。但是归根结底只有一片扇叶，为了避免误判，我们只在有两个moons或者一个monns一个t（并且这两个要基本在一条线上）的情况下进行更新。 度。、（因为一开始是这么想的，后面发现这个会导致状态更新混乱，就直接写了下面）
    • 如果 'moons' 中至少有一个moon。
  • 如果 fan_num 不等于 1 且 'moons' 中至少有一个moon

调试机制

这里的调试是在图像左上方显示的，可以通过调整d.SHOW_STATE_INFO来开启或关闭(yaml)

int k = 0

弹道模型与弹道补偿 一般情况下17mm弹丸发射角度接近水平，因此竖直方向速度分量极小，故忽略竖直方向空气阻力，仅考虑水平方向空气阻力对弹丸的作用。 设弹丸水平方向受到的空气阻力大小与速度平方成正比： \( |f| =kv_x^2 \)，则水平方向动力学模型为:

\[ -kv_x^2 = m\frac{\mathrm dv_x}{\mathrm dt} \]

根据初始条件 \(v_x(0) = v_{x|0}\) 解得：

\[ v_{x}(t)=\frac{v_{x|0}}{k_{1} v_{x|0} t+1} \]

其中 \(k_1 = \frac{k}{m}\)。水平速度 \(v_x(t)\) 对时间 \(t\) 积分得水平位移 \(x(t)\)：

\[ x(t) =\int_0^tv_{x}(\tau) \mathrm d \tau= \frac{1}{k_{1}} \ln \left(k_{1} v_{x|0} t+1\right) \]

根据位置估计值可得到目标水平距离 \(x\) 与水平速度 \(\dot x \)，则弹丸飞行时间 \(T\) 满足：

\[ x + \dot xT =\frac{1}{k_{1}} \ln \left(k_{1} v_{x|0} T+1\right) \]

利用牛顿迭代求解飞行时间 \(T\)，定义函数：

\[ f(t) =\frac{1}{k_{1}} \ln \left(k_{1} v_{x|0} t+1\right)-x - \dot xt \]

有：

\[ f^{'}(t) = \frac{v_{x|0}}{k_{1} v_{x|0} t+1}-\dot x \]

迭代公式为：

\[ \begin{align*} &T_0 = 0\\ &T_{k+1} = T_{k}-\frac{f(T_k)}{f^{'}(T_k)} \end{align*} \]

综上，云台应瞄准经过 \(T\) 秒后目标位置，即期望航向角 \(\psi\) 为：

\[ \psi = \tan^{-1}\frac{y_n + \dot y_nT}{x_n + \dot x_nT} \]
